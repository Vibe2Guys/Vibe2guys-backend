# Assignments Domain

## Core Entities

- Assignment
- Quiz
- Submission
- SubmissionAnswer
- GradeRecord

## Rules

- Assignments belong to a course.
- Assignments may be individual or team-based.
- Submission deadlines and late states must be represented explicitly.

## Supported Types

- multiple choice
- short answer
- descriptive answer

## Derived Outputs

- submission rate
- quiz score
- late submission count
- answer analysis summary

## Open Questions

- Will instructors manually grade descriptive answers in MVP?
- Are quizzes and assignments separate resources or one unified assessment model?
